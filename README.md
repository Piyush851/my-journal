**My Journal** is a React Native mobile application designed for daily journaling and mood tracking. Users can write journal entries, add emojis based on their mood, and keep track of their daily thoughts in a simple and intuitive way.

The app includes user authentication with Login and Signup functionality. Data is stored securely using MongoDB Atlas, with Express.js handling the backend APIs.
